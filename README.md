Название проекта

Книжный интернет-магазин "Маяк Знаний"

Авторы

Белогурова Анна, Дудла Олеся

Описание

Веб-сайт для продажи книг. Позволяет просматривать каталог, добавлять книги в корзину, изменять количество, удалять товары, оформлять заказ с контактными данными. Данные корзины сохраняются в `localStorage` браузера.

Функционал

- Главная страница с карточками книг (обложка, цена, кнопки «В корзину», «Подробнее»).
- Страницы деталей каждой книги.
- Корзина: добавление/удаление, изменение количества, автоматический подсчёт суммы, очистка.
- Оформление заказа: форма (ФИО, телефон, email, адрес) с валидацией, список товаров, итог, имитация подтверждения.

Технологии

HTML5, CSS3, JavaScript, Git/GitHub.
